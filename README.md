# Fumo Bird Game 🐦

Простая игра в стиле Flappy Bird на MonoGame + C#

## Что вам нужно установить

1. **Visual Studio 2022** (Community Edition - бесплатная)
   - https://visualstudio.microsoft.com/ru/vs/community/
   - При установке выберите: ".NET desktop development"

2. **MonoGame** (автоматически установится через NuGet)

## Как запустить

### Способ 1: Через Visual Studio
1. Откройте файл `FumoGame.csproj` в Visual Studio
2. Нажмите правой кнопкой → "Открыть в Visual Studio"
3. Нажмите F5 или Ctrl+F5 для запуска

### Способ 2: Через командную строку
```bash
cd c:\Users\Comp1\Desktop\fumogame
dotnet build
dotnet run
```

## Управление игрой

- **ПРОБЕЛ** или **Левый клик мышью** - прыгать
- **ESC** - выход

## Что есть в игре

✅ Основной персонаж (желтый квадрат) с гравитацией  
✅ Случайно генерируемые препятствия (зеленые трубы)  
✅ Система подсчета очков  
✅ Проверка столкновений  
✅ Главное меню  
✅ Game Over экран с лучшим результатом  
✅ Русский язык в интерфейсе  

## Структура кода

- **Player** - класс игрока с гравитацией и прыжком
- **Pipe** - класс препятствий
- **Game1** - основной класс игры с логикой
- **GameState** - перечисление состояний (Menu, Playing, GameOver)

## Как улучшить игру дальше

1. Добавить спрайты вместо квадратов
2. Добавить звуки
3. Добавить анимацию
4. Добавить частицы при столкновении
5. Сложность - увеличивать скорость с каждым уровнем
6. Добавить power-ups (щит, замедление времени)
7. Сохранение рекорда в файл
