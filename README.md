# Flappy Fumo 🐦

Игра в стиле Flappy Bird на MonoGame + C# с MVC архитектурой.

## Что нужно установить

1. **.NET 8 SDK**
   - https://dotnet.microsoft.com/download

2. **MonoGame Content Builder** (один раз, через терминал)
```bash
dotnet tool install -g dotnet-mgcb
```

## Как запустить

```bash
cd C:\Users\Comp1\Desktop\fumogame
dotnet run
```

## Управление

| Кнопка | Действие |
|--------|----------|
| Пробел / Левый клик | Прыжок |
| Tab | God Mode (вкл/выкл) |
| ESC | Выход |

## Что есть в игре

✅ Персонаж-фумо с гравитацией и прыжком  
✅ 3 типа труб: обычная (+1), движущаяся (+1), узкая (+3)  
✅ God Mode — розовая рамка по экрану, неуязвимость  
✅ Анимация фумо на экране проигрыша  
✅ Фон из BAKA BAKA на экране проигрыша  
✅ Фоновая музыка на меню и при проигрыше  
✅ Счёт и рекорд сессии  
✅ Русский язык в интерфейсе  

## Структура кода

```
FumoGame/
├── Models/          # Только данные, никакой логики
│   ├── PlayerModel.cs
│   ├── PipeModel.cs
│   └── GameModel.cs
├── Views/           # Вся игровая логика + отрисовка
│   └── GameView.cs
├── Controllers/     # Только приём ввода от игрока
│   └── GameController.cs
├── Content/         # Спрайты, шрифт, музыка
└── Program.cs       # Тонкая обёртка Game1 + точка входа
```

- **Models** — хранят данные (позиции, счёт, состояние), без логики
- **Views** — физика, коллизии, спавн труб, анимация, отрисовка
- **Controllers** — читают клавиатуру/мышь, передают команды во View
