# Flappy Fumo

Игра в стиле Flappy Bird на MonoGame + C# с MVC архитектурой.

## Что нужно установить

1. **.NET 8 SDK** — https://dotnet.microsoft.com/download

## Как запустить

Клонируй репозиторий, перейди в папку и запусти:

```bash
git clone https://github.com/kiralizanchuk/flappy-fumo.git
cd flappy-fumo
dotnet tool restore
dotnet run
```

`dotnet tool restore` установит MonoGame Content Builder автоматически — больше ничего ставить не нужно.

## Управление

| Кнопка | Действие |
|--------|----------|
| Пробел / Левый клик | Прыжок |
| Двойной клик | Рывок вперёд |
| Tab | God Mode (вкл/выкл) |

## Что есть в игре

- 3 типа труб: обычная (+1), движущаяся (+1), узкая (+3)
- Бонусы: щит, замедление, магнит, сердце, монеты
- Боссы каждые 60 очков со снарядами
- Тряска экрана, частицы при взаимодействии с бонусами
- Рывок по двойному клику 
- Выбор музыки (открывается после 3 смертей)

## Модульные тесты

Проект покрыт модульными тестами на xUnit — проверяются коллизии, модели игрока и босса, логика счёта и смертей.

```bash
dotnet test FumoGame.Tests/
```

После запуска увидишь результат вида `пройдено 33, не пройдено 0`.

## Структура кода

```
FumoGame/
├── Models/          # Данные (позиции, счёт, состояние)
├── Views/           # Логика, физика, отрисовка
├── Controllers/     # Ввод с клавиатуры и мыши
├── Content/         # Спрайты, шрифт, музыка
└── Program.cs
```
